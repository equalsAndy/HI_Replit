# #11 — IA 3-3: Unsplash search UX — clearer prompt, examples, and 'see more' option

**State:** OPEN
**Labels:** enhancement, content-update
**Created:** 2026-03-14

---

## Summary

In IA step 3-3 (Visualizing Your Potential), the Unsplash image search area has a few usability issues.

**Priority:** Medium
**Workshop:** IA
**Module:** 3, Step 3-3

## Issues

### 1. Search prompt doesn't stand alone
The PURPOSE paragraph above explains the concept well ("a side of yourself you haven't fully used yet"), but users skip straight to the activity section. The search prompt itself — "Find an image that represents something true about you that's waiting to be used" — is too vague without reading the purpose first.

**Fix:** Either repeat the key framing ("an underused capability / a side of yourself you haven't fully used") in the search prompt itself, or add a brief example to bridge the gap. Users shouldn't need to read the purpose block to understand what to search for.

### 2. Add an example
An example search/selection would help orient participants. Something like: "For example, if your underused capability is creativity, you might search for 'spark' or 'wild garden' — something that feels like that untapped energy."

### 3. No option to see more images or start over
Once Unsplash results load, there's no way to:
- See more results beyond the initial set
- Clear the search and start over with a new term

**Fix:** Add a "Show more" or "Load more" button and a "Clear / New search" option.

## Current Copy

**Purpose block:**
> Neuroscience shows the brain treats imagined experience almost like real experience — when your inner images shift, your choices shift with them. In this exercise, you'll find one image that captures a side of yourself you haven't fully used yet. Not just as a picture — as a starting point you'll come back to later.

**Search prompt:**
> Find an image that represents something true about you that's waiting to be used. It doesn't need to be literal — a wave, a spark, a mountain can all stand for something real.
